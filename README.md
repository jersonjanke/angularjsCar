# Angular Car

- AngularJs
- Javascript(ES6)
- Babel
- Webpack
- SASS
- John Papa

###### Clone
```
git clone https://github.com/jersonjanke/angularjsCar.git
```
###### Diretório
```
cd angularjsCar
```
###### Instalar dependâncias 
```
npm install
```

###### Start server 
```
npm start
```

###### Gerar build do projeto
```
npm run build
```

###### Start server a partir do build
```
npm run test-server
```

Acessar: [http://localhost:9000](http://localhost:9000)
